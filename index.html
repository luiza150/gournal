1. Загрузка оценок

В этой вкладке можно загрузить оценки из файла формата .csv .Выберите файл с данными и нажмите кнопку "Загрузить файл" . После загрузки файл будет отображен в таблице.

2. Редактирование журнала

На этой вкладке можно создавать, редактировать и удалять записи об учениках. Введите ФИО и оценки, затем используйте кнопки для управления записями.

3. Вывод (таблица)

Здесь табличная статистика для каждого класса и предмета, а также общая статистика для всех учеников. Представлены средняя оценка, медиана, количество и процент учеников для каждой оценки.

4. Вывод (график)

На этой вкладке можно визуально оценить статистику в виде графиков. Графики показывают среднюю оценку по классам
